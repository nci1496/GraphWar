# GraphWar 功能清单

## 1. 地图与据点

- 随机图生成，支持小/中/大地图模式。
- 据点类型：都城、城镇、村庄、关隘。
- 关隘不征兵、不收税，作为防御与交通节点。

## 2. 战斗与行军

- 派兵按批次出发，形成兵团流。
- 敌我相向兵团会触发道路遭遇战。
- 行军目的：占领、攻打、屠城、迁徙。

## 3. 资源与后勤

- 粮草/金钱都是据点本地库存。
- 支持运粮、运钱、销毁库存、修路、加固、维修。
- 断粮会导致城防归零，并持续累积不稳风险。

## 4. 发展线与数值约束（新版）

- 基础据点防御与屯兵：
  - 村庄：防御 `0/1`，屯兵上限 `15`
  - 城镇：防御 `1/2`，屯兵上限 `25`
- 经济线：
  - 防御上限：`2 -> 2 -> 3`
  - 屯兵上限：`30 -> 35 -> 40`
  - 高产粮、高产金
- 军事线：
  - 防御上限：`3 -> 5 -> 7`
  - 屯兵上限：`50 -> 70 -> 120`
  - 征兵效率高，升级后不自产粮，耗粮高
- 超出屯兵上限部分按 **10 倍驻军耗粮** 计费。

## 5. AI（规则评分重构）

- AI 独立模块：`graphwar/ai_logic.py`
- 目标函数：`argmax(动作后得分 - 当前得分)`
- 动作集：进攻、增援、运粮、运钱、修路、加固、升级
- 升级评估会考虑：
  - 交通要道/关隘/前线距离
  - 粮食压力与屯兵超限风险
  - 军事线与经济线配比（避免纯军事导致后勤崩溃）

## 6. 起义系统（新增）

- 支持 100% 出兵。
- 若空城且民心低于 60，高概率触发起义（绿色势力）。
- 周边低粮或低民心据点会高概率连锁起义。
- 起义军可独立扩张；与敌军接触后并入敌军（转红）。

## 7. 交互与调试

- 中文 UI 与棕色古地图风格。
- 右侧面板显示据点状态、屯兵上限和治理按钮。
- 开发者模式（F2）支持快速加兵/加人口测试。
